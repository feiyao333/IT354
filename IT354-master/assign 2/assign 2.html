Rest parameters

when we use '...something' in a parameter it gets bound to an array meaning: we get a list and can iterate over it. we think of this as having unlimited arguments.

function add(...numbers){
	let result =0;
	for (let number of numbers){
		result +=number;
	}
	return result;
}

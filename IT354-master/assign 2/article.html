ECMAscript

ECMAScript is the scripting language that forms the basis of JavaScript.

It standardized by the ECMA International standards organization

The first veriosn was released in 1997. And it was created in May 1995 and its other names were Mocha and LiveScript.

After the early days in ECMAScript, ECMAScript 4 was the abandoned specificaiton. In that time , ECMAScript 4 looks like a huge upgrade for ECMAScript 3. But it failed in that time. It can only released some of features.

The most important version is ECMAScript 6. Also, we called it ECMAScript 2015. It can be supported by the common browers, such as Chrome, Firefox, Ie.

There are some new features in ES6, such as let, const, arrow functions, default parameter, which we learnt in class.

If these features does not work, the reason may be is the worry version of ECMAScript.

In the ECMAScript 6, There are some useful tips for that, such as the swap value and merge function, which can increase coding more efficiently.

For instance, the swap replacing create a temp variable to switch two value of variables, which can help reducing the space complexity by omiting creating a new variable.
